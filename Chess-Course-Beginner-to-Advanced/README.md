# Complete Beginner to Advanced Chess Course

Welcome to your comprehensive journey to chess mastery. This course is designed to take you from knowing absolutely nothing about the game to understanding advanced strategic nuances.

## How to Use This Course
1.  **Follow the Study Plan**: Check `00-Study-Plan/` for a 4-week structured schedule.
2.  **Theory First**: Read the concept files in each module (e.g., `01-Basics`).
3.  **Active Practice**: Do the drills listed at the end of each lesson.
4.  **Solve Exercises**: Attempt the `exercises-XX.md` file *before* checking `solutions-XX.md`.
5.  **Review**: Use the spaced repetition checkpoints.

## Course Milestones

| Module | Skill Level | Goal / Capability |
| :--- | :--- | :--- |
| **01 Basics** | Absolute Beginner | Setup board, legal moves, checkmate in 1, basic notation. |
| **02 Openings** | Beginner | Survive the first 10 moves, control center, king safety. |
| **03 Tactics** | Intermediate | Spot 2-3 move tactical combinations (forks, pins). |
| **04 Strategy** | Intermediate+ | Evaluate positions, formulate simple plans. |
| **05 Middlegame** | Advanced Intermediate | Handle specific structures (IQP), attack kings. |
| **06 Endgames** | Advanced | Convert winning endgames, draw theoretical draws. |
| **07 Advanced** | Expert | Dynamic play, sacrifices, deep calculation. |

## Folder Structure
- **00-Study-Plan**: Your roadmap.
- **01-Basics**: Rules and fundamentals.
- **02-Opening-Foundations**: How to start the game.
- **03-Tactics**: Winning material and checkmates.
- **04-Strategy**: Long-term planning.
- **05-Middlegame-Planning**: Specific plans and structures.
- **06-Endgames**: Converting advantages.
- **07-Advanced-Topics**: Master-level concepts.
- **08-Assessment**: Test your skills.
- **09-Resources**: Books, tools, and further reading.
