# 02. Free Practice Methods

## 1. The "Woodpecker Method" (Lite)
- Take a set of 100 easy tactics.
- Solve them all.
- Do it again the next day.
- **Goal**: Pattern recognition.

## 2. Solitaire Chess
- Take a master game (without annotations).
- Cover the moves.
- Guess the winner's move.
- If you guess right, good.
- If wrong, ask "Why?" with an engine.

## 3. Blindfold Training
- try to visualize the color of squares (e.g., "What color is f6?").
- Try to calculate 3 moves deep in your head before touching a piece.
