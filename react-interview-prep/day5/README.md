# Day 5: Hooks Deep Dive II ✅

Welcome to Day 5! Today you'll master useRef, useMemo, and useCallback - and learn when NOT to use them!

## 📚 What You'll Learn

1. **useRef** - DOM access and stable mutable values
2. **useMemo** - Memoize expensive calculations  
3. **useCallback** - Memoize functions (when it actually helps!)

## 🗂️ Materials

- **[lesson.md](lesson.md)** - useRef, useMemo, useCallback explained
- **exercises/** - 3 hands-on exercises (45 mins total)
  - Exercise 1: useRef for DOM (focus, scroll, measure)
  - Exercise 2: useRef for Stable Values (render count, timers)
  - Exercise 3: useMemo & useCallback (optimization demo)
- **[interview-question.md](interview-question.md)** - Optimize searchable list
- **[mini-project.md](mini-project.md)** - Product Search with optimizations (60 mins)
- **[recap.md](recap.md)** - Summary and Day 6 preview

## ⏱️ Time Estimate

- Lesson: 15 mins
- Exercises: 45 mins
- Interview Q: 15 mins
- Mini-project: 60 mins
- **Total: ~2-2.5 hours**

## 🚀 Getting Started

1. Read `lesson.md` first
2. Complete exercises 1, 2, and 3 in order
3. Answer the interview question (optimize searchable list)
4. Build the Product Search mini-project
5. Review `recap.md`

## ✅ Success Criteria

By the end of Day 5, you should be able to:
- [ ] Use useRef for DOM access and stable values
- [ ] Understand when to use useMemo (and when NOT to)
- [ ] Use useCallback appropriately (with React.memo)
- [ ] Explain premature optimization problems
- [ ] Measure performance before optimizing
- [ ] Build performant search/filter interfaces
