# 📅 7-Day Beginner React Plan

## Day 1: React Fundamentals I
**Topics:** JSX, Components, Props

### Expected Outputs
- ✅ Understand what JSX is and how it differs from HTML
- ✅ Create 3 functional components from scratch
- ✅ Pass props between parent and child components
- ✅ **Project:** Build a `PersonCard` component that displays name, role, and bio

### Time Breakdown
- Lesson: 10 mins
- Exercises: 30 mins
- Interview Q: 10 mins
- Mini-project: 45 mins
- Review: 15 mins

---

## Day 2: React Fundamentals II
**Topics:** State, Lists, Keys

- ✅ Understand why keys are critical
- ✅ **Project:** Todo list (add items, display them)

### Time Breakdown
