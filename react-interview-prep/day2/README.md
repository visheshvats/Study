# Day 2: State, Lists & Keys ✅

Welcome to Day 2! Today you'll learn how to make React components truly interactive.

## 📚 What You'll Learn

1. **State Management** - Store and update data in your components
2. **Dynamic Lists** - Render arrays of data efficiently  
3. **Keys** - Why React needs them and how to use them correctly

## 🗂️ Materials

- **[lesson.md](lesson.md)** - 3-minute concept lesson covering useState, .map(), and keys
- **exercises/** - 3 hands-on exercises (40 mins total)
  - Exercise 1: useState Basics (Counter)
  - Exercise 2: Rendering Lists (BookList)
  - Exercise 3: State + Lists (ColorPicker)
- **[interview-question.md](interview-question.md)** - Keys in lists (why they matter)
- **[mini-project.md](mini-project.md)** - Build a dynamic Todo List (30-60 mins)
- **[recap.md](recap.md)** - Summary and Day 3 preview

## ⏱️ Time Estimate

- Lesson: 10 mins
- Exercises: 40 mins
- Interview Q: 10 mins
- Mini-project: 60 mins
- **Total: ~2 hours**

## 🚀 Getting Started

1. Read `lesson.md` first
2. Complete exercises 1, 2, and 3 in order
3. Answer the interview question
4. Build the Todo List mini-project
5. Review `recap.md`

## ✅ Success Criteria

By the end of Day 2, you should be able to:
- [ ] Create and update state using useState
- [ ] Explain the difference between props and state
- [ ] Render lists using .map() with proper keys
- [ ] Explain why keys are important for React's reconciliation
- [ ] Build an interactive component that manages its own state
