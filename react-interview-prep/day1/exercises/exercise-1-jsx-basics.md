# Exercise 1: JSX Basics (5 mins)

## 🎯 Goal
Write valid JSX without errors.

## 📝 Instructions

Create a file called `App.jsx` and fix the following broken JSX:

```jsx
function App() {
  return (
    <div>
      <h1>My First React App
      <img src="logo.png">
      <p class="intro">Welcome to React!</p>
    </div>
  )
}

export default App;
```

## ✅ Acceptance Criteria
- [ ] Code runs without errors
- [ ] All tags are properly closed
- [ ] Uses `className` instead of `class`
- [ ] Image tag is self-closing

## 🐛 Edge Cases
- What if you wanted to add multiple sibling elements? (Hint: Fragment `<>...</>`)

## 📋 Self-Review Checklist
- [ ] All JSX tags are closed
- [ ] No HTML attributes (only JSX equivalents)
- [ ] Code is properly indented

---

**Time limit:** 5 minutes  
**Difficulty:** ⭐☆☆

Once done, move to Exercise 2!
