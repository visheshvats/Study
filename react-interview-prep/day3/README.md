# Day 3: Events & Forms ✅

Welcome to Day 3! Today you'll learn how to handle user interactions and build real forms.

## 📚 What You'll Learn

1. **Event Handling** - Respond to clicks, typing, and more
2. **Controlled Components** - Build forms the React way  
3. **Conditional Rendering** - Show/hide UI based on state

## 🗂️ Materials

- **[lesson.md](lesson.md)** - 3-minute lesson on events, forms, and conditionals
- **exercises/** - 3 hands-on exercises (40 mins total)
  - Exercise 1: Event Handling (click, hover, keyboard)
  - Exercise 2: Controlled Components (registration form)
  - Exercise 3: Conditional Rendering (status display)
- **[interview-question.md](interview-question.md)** - Build a login form
- **[mini-project.md](mini-project.md)** - Counter + Contact Form (30-60 mins)
- **[recap.md](recap.md)** - Summary and Day 4 preview

## ⏱️ Time Estimate

- Lesson: 10 mins
- Exercises: 40 mins
- Interview Q: 15 mins
- Mini-project: 60 mins
- **Total: ~2 hours**

## 🚀 Getting Started

1. Read `lesson.md` first
2. Complete exercises 1, 2, and 3 in order
3. Answer the interview question (login form)
4. Build the mini-project (counter + contact form)
5. Review `recap.md`

## ✅ Success Criteria

By the end of Day 3, you should be able to:
- [ ] Handle events without infinite loops
- [ ] Build controlled form inputs
- [ ] Validate form data in real-time
- [ ] Use e.preventDefault() correctly
- [ ] Conditionally render UI with ternary and &&
- [ ] Disable buttons based on state
