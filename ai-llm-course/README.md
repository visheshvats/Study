# AI & LLM Engineering — From Beginner to Advanced

A comprehensive, practical course for software engineers transitioning into AI Engineering.

## Prerequisites
- Python 3.10+
- Java/Spring Boot background (helpful for analogies)
- OpenAI API key (or Ollama for local models)

## Modules

| # | Module | Phase |
|---|--------|-------|
| 0 | Foundations of AI and LLMs | Beginner |
| 1 | Prompt Engineering | Beginner |
| 2 | Embeddings & Vector Databases | Beginner |
| 3 | RAG (Retrieval Augmented Generation) | Intermediate |
| 4 | AI Agents (Core of Agentic AI) | Intermediate |
| 5 | MCP (Model Context Protocol) | Intermediate |
| 6 | Agentic AI Architecture | Advanced |
| 7 | Memory in Agents | Advanced |
| 8 | Tool Calling & Function Calling | Advanced |
| 9 | Build Production-Grade AI System | Expert |
| 10 | Advanced Topics (Fine-tuning, Local LLMs) | Expert |
| 11 | Deployment | Expert |

## Setup

```bash
cd ai-llm-course
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```
